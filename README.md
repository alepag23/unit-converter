# Unit Converter CLI

A simple CLI application in Go for converting units of measurement.

## Conversions

- km → miles
- kg → pounds
- Celsius → Fahrenheit

## Setup
```bash
mkdir unit-converter
go mod init unit-converter
touch main.go
```

## Usage
```bash
go run main.go
```

## Example Output
```text
23 km = 14.26 miles
100 kg = 220.50 lbs
30° C = 86.00 °F
```
